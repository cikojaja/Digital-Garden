# Personal Brand OS — by @cikojaja

A collection of Claude Code tools I built to create and grow a personal brand from scratch.
No agency. No budget. Just AI and 4AM mornings.

---

## What's Inside

### `brand-identity-skill.skill`
Generates a full Brand Identity HTML document for any brand.
Give it a URL (Instagram, website, Google Maps) → Claude researches the brand → outputs colors, fonts, audience, taglines, do's and don'ts.

**How to use:** Drag into Claude Code → drop any URL → type "build brand identity"

---

### `prompt-playbook.md`
Step-by-step workflow for building a personal brand with Claude.
Covers Instagram audit, niche finding, content strategy, carousel creation, and more.

**How to use:** Paste into Claude chat → follow the steps in order

---

### `NotebookLMSkill.md`
Automates Google NotebookLM — create notebooks, add sources, generate podcasts, infographics, and more.

**How to use:** Install as a skill in Claude Code → type `/notebooklm`

---

### `WrapUpSkill.md`
End-of-session skill that saves a summary of what was built, decided, and learned.

**How to use:** Install as a skill → type `/wrapup` at the end of any session

---

### `SKILL.md`
Meta skill for building new Claude Code skills. Use it to create your own.

---

## The Workflow

**Start here → follow in order:**

1. `brand-identity-skill.skill` — define your brand first
2. `prompt-playbook.md` — build your content system
3. `NotebookLMSkill.md` — research and generate assets
4. `WrapUpSkill.md` — save every session
