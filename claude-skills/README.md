# Personal Brand OS — by @cikojaja

A collection of Claude Code tools I built to create and grow a personal brand from scratch.
No agency. No budget. Just AI and 4AM mornings.

---

## What's Inside

### `brand-identity-skill.skill`
Generates a full Brand Identity HTML document for any brand.
Give it a URL (Instagram, website, Google Maps) → Claude researches the brand → outputs colors, fonts, audience, taglines, do's and don'ts.

**How to use:** Drag into Claude Code → drop any URL → type "build brand identity"

---

### `prompt-playbook.md`
Step-by-step workflow for building a personal brand with Claude.
Covers Instagram audit, niche finding, content strategy, carousel creation, and more.

**How to use:** Paste into Claude chat → follow the steps in order

---

### `NotebookLMSkill.md`
Automates Google NotebookLM — create notebooks, add sources, generate podcasts, infographics, and more.

**How to use:** Install as a skill in Claude Code → type `/notebooklm`

---

### `WrapUpSkill.md`
End-of-session skill that saves a summary of what was built, decided, and learned.

**How to use:** Install as a skill → type `/wrapup` at the end of any session

---

### `learning_skill.md` (Supportive Analogical Educator)
A universal teaching skill that instructs Claude to teach complex subjects, systems, or workflows using real-world analogies, visual flowcharts, and simple jargon-free language.

**How to use:** Install as a skill or prompt Claude to adopt this teaching style to learn any logic step-by-step.

---

### `SKILL.md`
Meta skill for building new Claude Code skills. Use it to create your own.

---

## The Workflow

Here is how the tools in this repository work together to help you build your personal brand:

### Data & Execution Flow:

```text
       [ Start with any Brand URL ] (Instagram, website, or social media link)
                    │
                    ▼
     1. brand-identity-skill.skill
        ├─ Analyzes the brand's style, colors, fonts, and tone
        └─ Creates a complete Brand Identity HTML Guidelines document
                    │
                    ▼
     2. prompt-playbook.md
        ├─ Finds your target niche and audience
        └─ Guides you step-by-step to build your content calendar and carousels
                    │
                    ▼
     3. NotebookLMSkill.md
        ├─ Connects with Google NotebookLM to build research sources
        └─ Automates creation of audio podcasts, summaries, and infographics
                    │
                    ▼
     4. WrapUpSkill.md
        ├─ Saves a summary of what you built, decided, and learned today
        └─ Prepares your next session tasks so you never lose momentum
                    │
                    ▼
     [ A Polished & Growing Personal Brand! ]
```

### Why use this order?
1. **Define First:** You cannot create content before knowing your brand's voice and colors (`brand-identity-skill.skill`).
2. **Strategy Second:** Once your style is set, build your plan and content templates (`prompt-playbook.md`).
3. **Research & Scale Third:** Gather deep knowledge and turn them into rich media like podcasts using AI (`NotebookLMSkill.md`).
4. **Log Daily:** Always save your progress so you can pick up exactly where you left off tomorrow (`WrapUpSkill.md`).
