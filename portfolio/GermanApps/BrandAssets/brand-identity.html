The potato mascot isn't random — it's charming and humble. Don't be corporate. Be fun.
Do's & Don'ts

Do

  • Use tonal depth — surface layers over shadows
  • Use pill shapes (border-radius: 999px) for buttons
  • Use #171d1e instead of pure black for text
  • Use the brand gradient for hero elements
  • Keep breathing room — generous padding
  • Use the potato mascot in marketing

Don't

  • Use 1px solid borders to divide sections
  • Use pure black (#000000) anywhere
  • Stack more than 3 surface container levels
  • Use German flag colors as primary UI elements
  • Use sharp corners — always round
  • Mention "AI" in any marketing copy
